## 🚀 Getting Started

### Installation 👨🏻‍💻

1. Install all packages

```
npm i
```

2. Run Development Server

```
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

3. Build your project

```
npm run build
```

4. Build preview

```
npm run preview
npm run serve
```
